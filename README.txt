** It only works with Pinta and in a 1920x1080 screen with its layout same as in screenshot.png **

** It only works with simple black and white images **

To run this you need --
    1) Python3
    2) pyAutoGUI
    3) Pinta
    4) and the arrangement as shown in screenshot.png
    5) Pinta's primary color to Black(#000000) and secondary color to White(#FFFFFF)
